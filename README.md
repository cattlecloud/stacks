Multiple implementations of a generic Stack data structure for optimizing by
varioius use cases.

```
stack (noun):
  1. a LIFO data structure
```

# Getting Started

The `stacks` package can be added to a Go project with `go get`.

```shell
go get noxide.lol/go/stacks@latest
```
